# Tic Tac Toe (TTT)

A browser-based Tic Tac Toe game built using JavaScript.

## 🎮 Features

- Two-player game
- Winner detection logic
- Reset functionality
- Real-time board updates

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript

## ▶ How to Run

1. Clone the repository  
2. Open `index.html` in your browser
